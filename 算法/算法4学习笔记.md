* 编译

````bash
$ javac -cp ~/Tools/crazyscript/algs4.jar Stats.java
````

错误: 找不到或无法加载主类 Stats，则删除 `package com.luci.chapter1_3;` package对应的语句。

* 运行

```bash
$ java Stats
```

错误: 找不到或无法加载主类 Stats，则删除 `package com.luci.chapter1_3;` package对应的语句。
